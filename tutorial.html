Cara Hosting Web di GitHub Pages

  1. Buat akun GitHub dan login.
  2. Buat repository baru (nama bebas).
  3. Upload file HTML dan folder CSS kamu.
  4. Masuk ke Settings > Pages.
  5. Pilih source: "main" dan root folder.
  6. Link web kamu muncul, misal: https://username.github.io/nama-repo/
